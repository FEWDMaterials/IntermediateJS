These are our three main objectives for today:

Game plan

Ok so here is what we will really be doing... OBJECTIVE: recreate jQuery.
WHY:

  • We use jQuery (most likely) every single day.
  • It is the single most popular javascript library on earth today (for now, anyways).
  • But what would happen if jQuery was inaccessible or we were not allowed to use it?

We will explore how the main objectives -- quite abstract principles -- come into play as we implement something quite abstract -- a library that handles DOM manipulation.
